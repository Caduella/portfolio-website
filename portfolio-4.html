Meet App

  • Category: React, Node, PWA
  • Project URL: Meet App
  • Project details: GitHub
  • Tech stack: React, Node, OAuth, Serverless, PWA, Jenkins, Gherkin, Jest, Cucumber, Puppeteer, AWS Lambda, Atatus

This is a serverless, progressive web application with React using a test-driven development technique. The application uses the Google Calendar API to fetch upcoming events. The app passed Lighthouse's PWA checklist and it was covered by tests with a coverage greater than 90%. The app was also monitored by Atatus (online performance monitoring tool).
